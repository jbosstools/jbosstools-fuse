Property

Overview

The property language provides a convenient way of accessing exchange properties. When you supply a key that matches one of the exchange property names, the property language returns the corresponding value.

Example

To implement the recipient list pattern when the listOfEndpoints exchange property contains the recipient list, you could define a route as follows:

<camelContext>
  <route>
    <from uri="direct:a"/>
    <recipientList>
      <property>listOfEndpoints</property>
    </recipientList>
  </route>
</camelContext>
