Creating a new chart

When it doesn't fit for the existing charts to have additional attributes, you may want to create a new chart.

To monitor additional attributes on a new chart:

  1. Open New Chart dialog by pressing New Chart button [] at local tool bar on Properties view.
  2. Add attributes in the same way as Configure Chart dialog described above.

Note: You can add a new chart also with Show in Timeline dialog that is available at MBeans page and Overview page.

Note: You can save the current set of charts with the local menu Save Chart Set As....
