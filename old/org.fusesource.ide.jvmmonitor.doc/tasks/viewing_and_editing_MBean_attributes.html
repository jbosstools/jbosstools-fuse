Viewing and editing MBean attributes

To view and edit MBean attributes:

  1. Select an MBean, and make sure that Attributes page is opened to view the associated MBean attributes.
  2. The MBean attribute icon with pencil [] indicates that the attribute is editable. If you want to edit attribute, slecect the table cell at Value column and edit it with table cell editor.

Note: If you want to monitor an attribute on Timeline page, select the attribute and select the context menu Show in Timeline....
